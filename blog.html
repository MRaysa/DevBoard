Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM in several ways like getElementById for unique elements, getElementsByClassName or getElementsByTagName for collections , querySelector or querySelectorAll for CSS selectors , getElementsByName for form elements with a name attribute and there are some DOM Traversal like parentElement,children,nextElementSibling,previousElementSibling etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main difference of this innerHTML is used to get or set HTML content, including elements and formatting. innerText gives only the visible text, ignoring hidden elements and treating everything as plain text. textContent shows all text, even hidden ones, without understanding HTML. In terms of speed, textContent is the fastest, while innerHTML is the slowest. So, use innerHTML for working with HTML, innerText for visible text, and textContent for getting raw text

Question-3: What is event delegation in the DOM?

Event delegation means adding an event to a parent element instead of each child element. When a child is clicked, the event bubbles up to the parent, which handles it. This makes the code faster, saves memory, and works even if new elements are added later.

Question-4: What is event bubbling in the DOM?

Event bubbling means when you click on a child element, the event first happens on the child, then moves up to its parent, grandparent, and so on. This lets parent elements react when their child is clicked.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript allows creating elements using document.createElement(), adding them with appendChild(), and removing them with remove(). This helps modify webpages dynamically.
